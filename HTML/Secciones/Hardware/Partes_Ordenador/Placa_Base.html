Placa base

¿Qué es?

La placa base (también llamada placa madre o motherboard) es uno de los componentes más importantes de un ordenador. Es el circuito principal que conecta y permite la comunicación entre todos los demás componentes del sistema: procesador (CPU), memoria RAM, tarjeta gráfica, almacenamiento, periféricos, etc.

¿Qué hace la placa base?

  • Distribuye la energía eléctrica a los componentes
  • Establece conexiones físicas y electrónicas entre componentes
  • Contine el chipset que gestiona el tráfico de datos entre la CPU, la memoria y otros dispositivos
  • Incluye puertos de entrada/salida (USB, audio, red, etc).
  • Tiene ranuras de expansión (PCIe) para tarjetas adicionales (gráfica, sonido, red, etc).

Tipos de placa base según el socket del procesador

El socket es el zócalo donde se instala el procesador. Cada generación o familia de CPU suele requerir un tipo específico.

  • Intel: LGA 1200, LGA 1700, LGA1151, etc.
  • AMD: AM4, AM5, TR4, etc.

Tipos de placa base según el chipset

El chipset determina las funciones que la placa puede ofrecer: cantidad de puertos, compatibilidad con overclocking, soporte para PCIe y almacenamiento rápido (NVMe, SATA, etc.).

  • Intel: H610, B660, Z790 (de menos a más avanzado)
  • AMD: A520, B550, X670

Factores de forma

Los factores de forma más comunes de las placas base utilizadas en PC son los siguientes:

ATX (Advanced Technology eXtended)

  • Tamaño: 30,5 x 24,4 cm
  • Uso: Ordenadores de escritorio estándar y gaming
  • Ventajas: Mayor número de ranuras de expansión (PCIe), Mejor distribución de componenetes, ideal para sistemas con múltiples tarjetas o buena ventilación

Micro-ATX (mATX)

  • Tamaño: 24,4 x 24,4 cm
  • Uso: PCs de oficina, domésticos, y económicos
  • Ventajas:Más compacta que ATX, pero todavía con suficientes ranuras, generalmente más económica
  • Limitaciones: Menor número de puertos y ranuras de expansión que ATX

Mini-ITX

  • Tamaño: 17 x 17 cm
  • Uso: PCs pequños, HTTPC (Home Theater PC, sistemas silenciosos o portátiles
  • Ventajas: Muy compacta, ideal para gabinetes pequeños
  • Limitaciones: Solo una ranura PCIe, menor capacidad para expansión y conectividad

E-ATX (Extended ATX)

  • Tamaño:30,5 x 33 cm o más
  • Uso: Workstations, PC de alto rendimiento, gaming extremo
  • Ventajas: Más espaciopara componentes de gama alta (doble GPU, más RAM, etc)
  • Limitaciones: Requiere una caja grande, precio más elevado
